── LEADER (Ctrl+K then...) ──────────────────────────────────

SPLITS
  \        Split right
  -        Split down

PANE NAVIGATION
  h/j/k/l  Navigate panes (vim directions)
  ←↑↓→     Navigate panes (arrows)
  o        Focus next pane

PANE MANAGEMENT
  x        Close pane
  z        Toggle zoom/fullscreen
  =        Toggle zoom/fullscreen (alias)
  w        Toggle floating panes

TAB MANAGEMENT
  t        New tab
  e        Rename tab
  ,        Rename tab (alias)
  n        Next tab
  p        Previous tab
  Tab      Toggle last tab
  1-9      Jump to tab by number

SUB-MODES
  r        Enter resize mode (then h/j/k/l to resize)
  m        Enter move mode (then h/j/k/l to move panes)

COPY/SCROLL
  c        Enter scroll mode
  [        Enter scroll mode (alias)

SESSION
  s        Session manager (floating plugin)
  d        Detach from session

MISC
  Space    Next swap layout
  Ctrl+K   Send literal Ctrl+K (passthrough)
  ?        Keybinding help (zellij-forgot plugin)

── SUB-MODES (entered via Ctrl+K or global) ─────────────────

PANE MODE (Alt+P)
  h/j/k/l  Navigate panes
  ←↑↓→     Navigate panes (arrows)
  n        New pane
  d        Split down
  r        Split right
  x        Close pane (tab mode only)
  c        Rename pane
  e        Toggle embed/floating
  f        Toggle fullscreen
  i        Toggle pane pinned
  w        Toggle floating panes
  z        Toggle pane frames
  p        Switch focus
  Ctrl+P   Exit pane mode

TAB MODE (Alt+T)
  h/l      Previous/Next tab (vim)
  j/k      Next/Previous tab (vim)
  ←→↑↓     Navigate tabs
  n        New tab
  r        Rename tab
  x        Close tab
  s        Toggle sync tab
  b        Break pane to new tab
  [        Break pane left
  ]        Break pane right
  1-9      Jump to tab by number
  Tab      Toggle last tab
  Ctrl+T   Exit tab mode

RESIZE MODE (Ctrl+K r)
  h/j/k/l  Increase size (direction)
  H/J/K/L  Decrease size (direction)
  ←↑↓→     Increase size (direction)
  +/=      Increase size
  -        Decrease size
  Ctrl+N   Exit resize mode

MOVE MODE (Ctrl+K m)
  h/j/k/l  Move pane (direction)
  ←↑↓→     Move pane (direction)
  n/Tab    Move pane (next)
  p        Move pane backwards
  Ctrl+H   Exit move mode

SCROLL MODE (Ctrl+K c or [)
  j/k      Scroll down/up (line)
  d/u      Scroll down/up (half page)
  h/l      Scroll up/down (page)
  Ctrl+F/B Page down/up
  ←→↑↓     Scroll (arrows)
  PageUp/Dn Page scroll
  e        Edit scrollback in $EDITOR
  s        Enter search mode
  Ctrl+C   Scroll to bottom and exit
  Ctrl+S   Exit scroll mode

SEARCH MODE (from scroll: s)
  n        Search next
  p        Search previous
  c        Toggle case sensitivity
  w        Toggle wrap
  o        Toggle whole word

SESSION MODE (no global shortcut — use Ctrl+K s)
  w        Session manager
  c        Configuration plugin
  p        Plugin manager
  a        About
  Ctrl+O   Exit session mode

── GLOBAL (no leader needed) ─────────────────────────────────

NAVIGATION
  Alt+H/L     Move focus or switch tab (left/right)
  Alt+J/K     Move focus (down/up)
  Alt+←→↑↓    Move focus or switch tab (arrows)
  Alt+[/]     Focus previous/next pane
  Alt+1-9     Jump to tab by number

PANES
  Alt+N       New pane
  Alt+F       Toggle floating panes

TABS
  Alt+I       Move tab left
  Alt+O       Move tab right

RESIZE
  Alt+ +/=    Increase size
  Alt+ -      Decrease size

LAYOUT
  Alt+{       Previous swap layout
  Alt+}       Next swap layout

MODES
  Ctrl+G      Toggle locked mode (all keys pass through)
  Ctrl+Q      Quit Zellij

SHARED
  Enter       Return to normal mode (from any sub-mode)
  Esc         Return to normal mode (from most sub-modes)

── CLI (for coding agents) ───────────────────────────────────

  zellij action rename-tab "NAME"
  zellij action rename-pane "NAME"
  zellij action new-tab --name "NAME"
  zellij action new-tab --layout path/to/layout.kdl
