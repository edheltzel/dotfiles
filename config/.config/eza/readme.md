# eza Theme Configuration

[eza](https://github.com/eza-community/eza) looks for `theme.yml` in `~/.config/eza/`.

## Usage

To switch themes, copy one from `themes/` to `theme.yml`:

```bash
cp ~/.config/eza/themes/eldritch.yml ~/.config/eza/theme.yml
```

## Available Themes

- `default.yml`
- `eldritch.yml`
- `one_dark.yml`
- `rose-pine.yml`
- `rose-pine-dawn.yml`
- `rose-pine-moon.yml`
- `tokyonight.yml`
